# Ring Rhythm

Minimal browser rhythm game prototype built with React and Vite.

## Flow

1. Title screen
2. Setup screen: upload audio and generate a basic chart
3. Play screen: play the 4K downscroll stage

## Run

Install dependencies first:

```bash
npm install
```

Then start the dev server:

```bash
npm run dev
```

Open the Vite URL shown in the terminal.

## Features

- English UI
- Audio upload
- Basic chart generation from audio features
- 4K downscroll playfield
- Tap-note gameplay
- React component structure

## Structure

- `TitleScreen`
- `SetupScreen`
- `PlayScreen`
- `SettingsModal`
- `useRhythmGame`
